
# Python_Games_Demo_Project

A collection of fun, beginner-friendly console-based games written in Python. Perfect for learning Python basics like loops, conditionals, functions, and user input handling.

## Games Included

1. **Hangman**  
   Guess the word letter by letter before the hangman is fully drawn!

2. **Number Guessing**  
   The computer picks a random number — can you guess it within limited attempts?

3. **Odd or Even**  
   Play against the computer: pick odd or even, roll the dice, and see who wins!

4. **Vowel or Consonant**  
   A letter is randomly chosen — guess whether it's a vowel or a consonant!

---

## Features

- Simple and clean code
- Beginner-friendly with comments
- No external dependencies (uses only Python standard library)
- Interactive console interface
- Random elements for replayability

## Requirements

- Python 3.6 or higher
